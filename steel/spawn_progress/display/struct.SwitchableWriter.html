Struct SwitchableWriter

pub struct SwitchableWriter {
    inner: Arc<SyncMutex<Option<SpawnProgressDisplay>>>,
}
Expand description

A tracing writer that can redirect output through a SpawnProgressDisplay.

When the display is not activated, output goes directly to stderr. When activated, log lines are rendered above the progress grid.

Internally reference-counted โ cloning is cheap and shares the same state.
