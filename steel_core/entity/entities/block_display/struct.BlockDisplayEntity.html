    base: EntityBase,

Expand description

A block display entity that renders a block state at its position.

Block displays are purely visual entities with no collision. They support transformation (translation, rotation, scale) and interpolation for smooth animations.

pub fn from_saved( id: i32, position: Vector3<f64>, uuid: Uuid, world: Weak<World>, ) -> Self

Creates a block display entity from saved data.

Display entities don’t use velocity, rotation, or on_ground, so this is essentially an alias for with_uuid. Type-specific data is restored via load_additional() after construction.
