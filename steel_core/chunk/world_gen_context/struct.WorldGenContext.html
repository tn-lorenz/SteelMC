Struct WorldGenContext

pub struct WorldGenContext {
    pub generator: Arc<ChunkGeneratorType>,
    world: Weak<World>,
}
Expand description

Context for world generation.

Similar to vanilla’s WorldGenContext, this provides access to the level/dimension and generation infrastructure.

Fields§

§generator: Arc<ChunkGeneratorType>

The chunk generator to use.

§world: Weak<World>

Weak reference to the world (to avoid circular Arc reference). Use world() to get a strong reference when needed.
