Expand description

Barrel block entity.

A simple container with 27 slots, using the same menu as chests.

Fields§

§level: Weak<World>

Weak reference to the world for marking chunks dirty.

§pos: BlockPos

Position in the world.

§state: BlockStateId

Current block state.

§removed: bool

Whether this entity has been marked for removal.

§items: Vec<ItemStack>

The 27 item slots.
