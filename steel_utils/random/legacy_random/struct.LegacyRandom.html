Struct LegacyRandom

pub struct LegacyRandom {
    seed: i64,
    next_gauissian: Option<f64>,

Expand description

Legacy Minecraft random number generator based on a Linear Congruential Generator (LCG). This implementation mirrors Java’s java.util.Random which Minecraft originally used.

pub fn from_seed(seed: u64) -> Self

Creates a new LegacyRandom instance from the given seed. The seed is XORed with the LCG multiplier and masked to 48 bits, matching Java’s behavior.
